# Changelog

All notable changes to this project will be documented in this file.

The format follows [Keep a Changelog](https://keepachangelog.com/) and uses semantic versioning.

## [1.0.0] - 2026-03-29

### Added
- Five-question model selection flow for OpenClaw use cases.
- Opinionated primary and fallback model recommendations based on use case, budget, billing style, tool-call needs, and local-model tolerance.
- Free-tier and local-model branches for zero-cost or self-hosted paths.
- Paste-ready configuration snippet generation for the recommended setup.
- Monthly cost range estimates for the chosen stack.
- Shareable result text and URL-encoded state for easy reuse and discussion.
- Calculator handoff link so users can verify the exact spend of a chosen stack.
- OpenRouter warning for routing-layer recommendations.
- GuardClaw ecosystem landing page, branding, and supporting explanatory content.

### Changed
- Moved model selection away from guesswork and toward a guided decision flow.
- Normalized recommendations into a structured primary/fallback stack instead of isolated model suggestions.
- Framed model choice around reliability, budget, and deployment fit rather than raw capability alone.

### Fixed
- Reduced trial-and-error model selection for common OpenClaw workflows.
- Eliminated the need to manually assemble a config after choosing a model.
- Clarified the cost implications of subscription, API, and free-tier paths.

### Removed
- No prior shipped release features to remove in the initial version.

### Deprecated
- No deprecated behavior in the initial version.
